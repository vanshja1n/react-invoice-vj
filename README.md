# InvoiceHub

A modern **Invoice & Inventory Management System** built with **React, Vite, Tailwind CSS, and IndexedDB**. Manage customers, products, inventory, and invoices with professional PDF generation, analytics dashboard, and offline-first local storage.

---

## ✨ Features

### 📊 Dashboard
- Business analytics overview
- Monthly revenue tracking
- Recent invoices
- Recent customers
- Low stock alerts
- Invoice status summary

### 🧾 Invoice Management
- Create, edit and delete invoices
- Duplicate invoices
- Invoice status tracking
- Professional PDF generation
- Print-ready invoices
- ZIP export of invoices
- Multiple invoice templates
- Corporate template as default

### 👥 Customer Management
- Add, edit and delete customers
- Customer purchase history
- Customer search
- Recent customer activity

### 📦 Product & Inventory Management
- Product catalog
- SKU support
- Categories
- Cost & selling price
- Current stock tracking
- Low stock alerts
- Automatic inventory updates

### ⚙️ Settings
- Business profile
- Company logo
- GST/VAT configuration
- Currency selection
- Default invoice settings
- Theme customization

### 💾 Offline First
- IndexedDB (Dexie.js)
- Local data persistence
- Backup & Restore
- JSON Export/Import

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/yourusername/invoicehub.git
```

Go into the project

```bash
cd invoicehub
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
